## What it does

`nuxt-feathers-zod` is a Nuxt module for **FeathersJS v5** that supports two main workflows:

- **embedded mode**: run a Feathers server inside Nuxt/Nitro
- **remote mode**: connect a Nuxt app to an external Feathers API

It also provides:

- typed service generation
- CLI-first scaffolding
- auth helpers
- optional Zod-first workflow
- MongoDB management support
- Pinia / client helpers
- documentation and playground
