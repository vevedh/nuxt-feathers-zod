# MASTER PROMPT

We are continuing development of the Nuxt module:

nuxt-feathers-zod.

The module integrates FeathersJS v5 into Nuxt 4 with Zod-first schemas.

Architecture supports:

embedded mode (Feathers inside Nitro)
remote mode (Nuxt client connecting to external API)

The module provides:

CLI generators
server modules
middleware system
VitePress documentation

Goal:

Create a DevOps-ready framework combining:

Nuxt + Feathers + CLI tooling + DevTools + Builder UI
